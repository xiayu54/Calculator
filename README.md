# Calculator
A simple calculator：
This is simple Calculator;

分为3块：
1、UI界面
2、逻辑代码-MainActivity
3、逻辑算法-TransToDeal

实现了如下功能：
1.基本的加减乘除
2.点击按钮时，按钮会变色，让使用者可以知道自己点的是哪个按钮，点了几次
3.可以记住上次计算的结果
4.错误输入时会有提示
